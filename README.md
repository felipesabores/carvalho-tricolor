# 🤍❤️💚 Carvalho Tricolor

> *"Nenhum Passo Atrás."*

App mobile-first exclusivo da **Família Carvalho** — torcedores de coração do maior clube do Brasil, o **Fluminense Football Club**.

---

## Do que se trata

O Carvalho Tricolor nasceu de uma paixão que atravessa gerações. Enquanto o mundo inteiro assiste, a Família Carvalho **vive** cada jogo, cada gol, cada conquista com o Fluzão. Este app é o nosso cantinho digital — exclusivo, fechado e tricolor.

Nada de feed genérico. Nada de notícia de time de outro. Aqui é só **Flu**.

---

## As Telas

| Tela | O que faz |
|------|-----------|
| **Login** | Acesso exclusivo. Só entra quem é da família. |
| **Home Dashboard** | Próximo jogo em destaque, alertas e as últimas do Fluzão. |
| **Agenda da Família** | Calendário completo com datas, horários, canais e competições. |
| **Tempo Real** | Placar ao vivo, timeline de lances e estatísticas do jogo. |
| **Leitura de Notícia** | Artigo completo com a notícia do Tricolor das Laranjeiras. |
| **Meu Perfil** | Preferências de notificação e dados do membro da família. |

## Design System

Segue fielmente a identidade visual oficial do **Fluminense FC**:

```
Grená:    #6E182C   → headers, destaques, títulos
Verde:    #33603B   → botões de ação, CTAs
Fundo:    #FFFFFF
Texto:    #727175
Links:    #01633D
Fonte:    Open Sans (400 / 600 / 700 / 800)
```

---

## Stack

- HTML5 + CSS3 + Vanilla JS
- PWA (Progressive Web App) — mobile-first, sem frameworks
- Zero dependências de runtime

---

## Como rodar localmente

```bash
# Com Python (sem instalar nada)
python -m http.server 3000

# Ou com Node
npx serve .
```

Acesse `http://localhost:3000` e entre com as credenciais da família.

---

## História

Criado em março de 2026, temporada em que o Flu briga pelo Carioca e se prepara para mais uma campanha na Libertadores. A Família Carvalho precisava de um app à altura do clube — simples, bonito e tricolor.
